Vegetable Service Engine

  • POST /vegetable-service-engine/delete?vegetableName=...
  • POST /vegetable-service-engine/calcost?vegetableName=...&quantity=...
  • POST /vegetable-service-engine/receipt?totalCost=...&amountGiven=...&cashierName=...
